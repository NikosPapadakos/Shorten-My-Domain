api -> Περιέχει το Rest API που χρησιμοποιείται για να τελέσει τις διάφορες ενέργειες της εφαρμογής.

class -> Περιέχει τα classes για το api.

config -> Περιέχει το class της βάσης (mySQL).

misc -> Περιέχει την εξαχθείσα βάση.

scripts -> Περιέχει την λογική των διάφορων σελίδων σε μορφή javascript.

styles -> Περιέχει αρχεία png και css για την μορφοποιήση των σελίδων.

.htaccess -> Δημιουργεί shortcut για την σμίκρυνση των παραγόμενων url.
!ΠΡΟΣΟΧΗ! Αν τα παραγώμενα urls (μορφής  http://localhost/smd/slug) δεν δουλεύουν το πραγματικό link ειναι:
http://localhost/smd/redirect.php?shortened=slug οπου slug ειναι ο παραγώμενος 
επταψήφιος κωδικός που ζητάει η άσκηση.

admin.php -> Η διαχειριστική σελίδα.

expiredAndRenew.php, redirect.php, urlExpired.php -> Σελίδες που αφορούν την ανακατεύθυνση από
συμπιεσμένο url.

login.php -> Η σελίδα εισόδου διαχειριστή.

index.php -> Η αρχική σελίδα της εφαρμογής.
